# GA6短信指南
### 首先你要确定你的硬件都连接好了
### 先用AT指令测试一下
**休息:串口助手上不能勾选加回车换行**  
发送  

	AT\r
结果  

	OK  
##GSM短信发送模式##

### 1、设置GSM短信模式###
发送  

  AT+CMGF=1\r
结果  

  OK
### 2、设置短信编码格式###
发送  
  
  AT+CSCS="GSM"\r
结果  

  OK
### 3、设置短信号码###
发送  

  AT+CMGS="13453458521"\r
结果  

  >
### 4、发送数据###
发送  

  test\r
结果  

  +CMGS: 1
  
  OK
