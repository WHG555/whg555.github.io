# 介绍 #  
树状控件（QTreeWidget）用于表示有树状层次关系的数据，该控件的每一项均可添加位图或文字，可响应单击、双击、选项改变、树状显示扩展、收缩等信号。   
# 所需头文件 #  

	#include <QTreeWidget>
	#include <QTreeWidgetItem>
