# 配置 #
```
{
    "configurations": [
        {
            "name": "Python: 当前文件",
            "type": "python",
            "request": "launch",
            "program": "${file}",
            // python 包的位置
            "pythonPath": "E:\\soft\\Anaconda3\\envs\\py38\\python.exe",
            "console": "integratedTerminal"
        }
    ]
}
```
